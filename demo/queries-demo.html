Solid UIX home | keywords | basic queries | advanced queries
query for a table
<table data-uix="query"
    data-select="name age email"
    data-from="test-data.ttl"
    data-where="type Person"
></table>
query for a record
<div data-uix="query"
    data-from="test-data.ttl#Thing1"
></div>
query for a value
<b data-uix="query"
    data-select="type"
    data-from="test-data.ttl"
    data-where="name Fido"
></b>

fill templates from queries
<div data-uix="query"
    data-select="name age"
    data-from="test-data.ttl"
    data-template=
     "<p>${name} is ${age} years old.</p>"
></div>

fill built-in interactive templates (tabs,accordions,etc.) from queries
<div data-uix="query"
    data-select="name"
    data-distinct="name"
    data-from="test-data.ttl"
    data-template="Tabset"
><div>
